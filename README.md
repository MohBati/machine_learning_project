**Project Workflow**

**1. Data Preprocessing**
Cleaned and prepared the dataset
Handled missing values
Encoded categorical variables
Scaled/normalized numerical features
Split data into training and testing sets

**2. Exploratory Data Analysis (EDA)**
Analyzed feature distributions
Visualized relationships between variables
Identified patterns and potential correlations
Generated insights from the dataset

**3. Model Training**
Trained machine learning models using the processed data
Compared model performances
Evaluated accuracy and other performance metrics

**4. Model Testing & Evaluation**
Tested models on unseen data
Measured model performance using evaluation metrics
Analyzed strengths and weaknesses of the models

**5. Hyperparameter Tuning**
Applied basic tuning techniques to improve model performance
Due to hardware limitations and training time requirements, tuning was not fully optimized
Future improvements may include deeper tuning and testing on more powerful hardware
